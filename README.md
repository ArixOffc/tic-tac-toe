## Cara Instalasi

1. Clone repositori ini:
   ```bash
   git clone https://github.com/ArixOffc/tic-tac-toe.git
   cd tic-tac-toe
   ```
2. Install dependensi:
   ```bash
   npm install
   ```
3. Jalankan server:
   ```bash
   npm start
   ```
4. Buka di browser: [http://localhost:3000](http://localhost:3000)
